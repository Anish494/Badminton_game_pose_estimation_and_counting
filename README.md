"# Badminton Shot Classifier" 
A real-time badminton shot classification system using YOLOv8 pose estimation to detect and categorize badminton strokes from video input.

## Features
- Detects and classifies 4 shot types: Forehand, Backhand, Smash, and Serve
- Uses YOLOv8 pose estimation to track player body keypoints
- Outputs shot counts and results to CSV

## Tech Stack
- YOLOv8 (Ultralytics)
- Python
- OpenCV
- Pandas

## Files
- `Badminton_shot_classifier.ipynb` — main notebook with model and classification logic
- `shot_results.csv` — sample output of shot counts
